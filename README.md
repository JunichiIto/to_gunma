# ToGunma

A Gem for learning.

See also: http://soplana.hateblo.jp/entry/2012/06/01/234259

## Installation

Add this line to your application's Gemfile:

    gem 'to_gunma'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install to_gunma

## Usage

````
$ pry
> require 'to_gunma'
=> true
> ''.to_gunma
=> "String is Gunma now."
> :hoge.to_gunma
=> "Symbol is Gunma now."
````
